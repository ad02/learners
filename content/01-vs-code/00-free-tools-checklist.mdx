# Start Here — Free Tools Checklist

Before you dive in, here's the honest truth: **this course uses real tools, not just simulators.** By the end, you'll have actual working skills — not just trivia.

The good news? **Every single tool is free.** No credit card required, no trial that expires, no surprise bills. We picked tools specifically because they have generous free tiers.

This page is your **one-stop reference** — bookmark it. Every tool you'll need is listed below with a signup link. You don't need to create everything today. Each module will tell you when to sign up for what.

## The Golden Rule

> **If anything in this course tells you to pay for something, stop and tell us.** Everything here works on free tiers. If you hit a paywall, you're probably on the wrong signup page.

---

## 🗂️ Tools by Module

### Module 1 — VS Code (Your Code Editor)

**Tool:** [Visual Studio Code](https://code.visualstudio.com)
**What it is:** A free app for writing code. Like Microsoft Word but for programming.
**Free forever?** Yes. Made by Microsoft, no account needed.
**What you'll do:** Install it on your computer.

---

### Module 2 — Git & GitHub (Saving Your Work)

**Tool 1:** [Git](https://git-scm.com/downloads) (the program)
**What it is:** A tool that saves versions of your files so you never lose work.
**Free forever?** Yes. Open source.

**Tool 2:** [GitHub](https://github.com/signup) (the website)
**What it is:** Online home for your code. Like Google Drive but for programmers.
**Free tier:** Unlimited public AND private repositories. You'll never outgrow it.
**What you'll do:** Create an account, connect it to Git on your computer.

---

### Module 3 — Prompting (Talking to AI)

**Tool:** [Claude.ai](https://claude.ai) (free tier)
**What it is:** A chat interface to Claude, an AI assistant.
**Free tier:** Several messages per day, enough to practice. No credit card.
**What you'll do:** Sign up, practice writing good prompts.

**Backup option:** [ChatGPT free](https://chat.openai.com) — works too if Claude.ai is slow.

---

### Module 4 — Claude Code (AI in Your Terminal)

**Tool:** [Claude Code](https://claude.com/claude-code) (CLI)
**What it is:** An AI coding assistant that runs in your terminal.
**Free tier:** Limited free usage via Claude.ai account.
**What you'll do:** Install it, have your first AI-assisted coding session.

> **Heads up:** Claude Code may ask for a paid API key for heavy use. For this course, the free tier is plenty.

---

### Module 5 — Web Basics (HTML/CSS/JS)

**Tool:** Just your browser + VS Code (already installed).
**What it is:** You'll write HTML files and open them in Chrome/Firefox/Edge.
**Free?** No new accounts needed. 🎉

---

### Module 6 — WordPress

**Tool:** [WordPress.com](https://wordpress.com/start) (free plan)
**What it is:** A free website hosted by WordPress themselves.
**Free tier:** Your own site at `yourname.wordpress.com`. Ads shown, but fine for practice.
**What you'll do:** Create a free blog, explore the dashboard.

> **Not WordPress.org** — that version requires you to buy hosting. Use **WordPress.com** for this course.

---

### Module 7 — APIs & Webhooks

**Tool 1:** [Groq](https://console.groq.com/signup) (free AI API)
**What it is:** Free API access to Llama models. Lightning fast.
**Free tier:** Very generous — more than you'll use for practice. No credit card.
**What you'll do:** Get an API key, make your first real API call.

**Tool 2:** [Postman](https://www.postman.com/downloads/) (free API tester)
**What it is:** A desktop app for testing APIs without writing code.
**Free tier:** Forever free for individual use.
**What you'll do:** Send your first real API request.

**Tool 3:** [Webhook.site](https://webhook.site) (no signup needed)
**What it is:** A website that gives you a temporary URL to receive webhooks.
**Free?** Yes, no account. Just visit and use.

## 🔒 Keeping Your Keys Safe

Later in the course, you'll get **API keys** — these are like passwords. A few rules from day one:

1. **Never paste API keys into a public GitHub repo.** People's bots scrape these and steal credits.
2. **Never share your API key in screenshots.** Blur it out.
3. **If you leak a key, rotate it.** Every service has a "revoke / regenerate key" button.

Don't worry — we'll cover this again when you get your first real key in Module 7.
