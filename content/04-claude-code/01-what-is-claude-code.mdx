# What is Claude Code?

Imagine having a **senior developer sitting right next to you** — someone who never gets tired, never judges your questions, and is always ready to help. That's Claude Code.

Claude Code is an **AI assistant that lives in your terminal**. Instead of chatting in a browser window, you talk to it right where you write code. It can see your files, understand your project, and help you build things.

## What Can It Do?

Claude Code is like having a super-powered teammate. It can:

- **Create files** — ask it to make a new page, component, or config file
- **Write code** — describe what you want and it writes it for you
- **Fix bugs** — show it an error and it figures out the fix
- **Explain code** — point it at confusing code and it breaks it down in plain English
- **Answer questions** — ask it anything about your project or programming in general

## How Is It Different from ChatGPT?

You might have used ChatGPT or Claude in a web browser before. Claude Code is different because:

1. **It sees your project** — it can read all your files and understand how they fit together
2. **It makes changes directly** — instead of copying and pasting, it edits your files for you
3. **It lives in your terminal** — no switching between browser tabs and your code editor

Think of it this way: chatting with AI in a browser is like texting a friend for advice. Using Claude Code is like having that friend **sit down at your computer** and help you hands-on.

## Do I Need to Be a Programmer?

Not at all! That's the whole point. You describe what you want in plain English, and Claude Code does the technical work. As you use it more, you'll naturally start to understand the code it writes — but you don't need to know any programming to get started.
