# Why Learn About the Web?

You might be thinking: "I signed up to learn AI automation — why do I need to learn about websites?" Great question!

## Almost Everything Touches the Web

Here's the thing: **almost every automation you'll build connects to the web** in some way. Filling out forms, pulling data from websites, talking to APIs, sending emails — it all happens through the web.

If you don't understand how the web works, automation tools will feel like magic. And magic is hard to fix when it breaks.

## The Driving Analogy

Think of it like this: **learning the web is like learning to drive before you can deliver packages.**

You don't need to be a mechanic. You don't need to understand how an engine works internally. But you do need to know how to steer, where the gas pedal is, and how to read road signs.

That's what this module is about — giving you just enough knowledge to navigate confidently.

## What You'll Learn

In this module, we'll cover:

- **What websites actually are** (it's simpler than you think!)
- **HTML basics** — the building blocks of every webpage
- **Domains and hosting** — how websites get their names and where they live

## Everything Else Will Click

Once you understand these basics, everything we do later — connecting to APIs, automating web tasks, building workflows — will make so much more sense.

You'll go from "I have no idea what's happening" to "Oh, that's just a website sending data to a server!" And that shift changes everything.

Let's get started!
